# Alexandria
Content API server that responds to GraphQL by providing content for pages, posts, etc

Base objects that contain required fields for their type.

# Site
``{
  "title": "title",
  "theme": "basic"
  }``

# Page
``{
  "title": "title"
  }``

# Post
``{
  "title": "title"
  }``

# Product
``{
  "name": "widget"
  }``

#Person
``{
  "name": {
      "given": "David",
      "family": "Smith"
    }
  }``
